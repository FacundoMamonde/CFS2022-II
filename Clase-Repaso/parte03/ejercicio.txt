• Enumerar tres funcionalidades de NPM y describirlas
    1-Descargar librerias Js
    NPM Nos permite descargar librerías que nos pueden facilitar el trabajo a la hora de armar un projecto, tambien permite actualizar estas librerias a una version mas nueva

    2-package.json 
    En el archivo package podemos establecer que librerias queremos descargar y en que version deseamos hacerlo, podemos conservar este archivos para volver a utilizarlo en algun proyecto similar

    3-Compartir
    Podemos compartir nuestras propias librerias al resto de usuarios
